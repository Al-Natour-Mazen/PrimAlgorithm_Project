	* Un dossier src :
		Contenant les fichiers sources de notre projet.

	* Un makefile :	
		Il s'agit ici de plusieurs commandes afin de faciliter la compilation et l'exécution de notre programme, ainsi
	  vous avez à votre disposition les commandes suivantes :
		
		=> make ou make all :
			Permettant de compiler le programme.
		=> make run :
			Permettant de compiler le programme et de le lancer.
		=> make clean :
			Permettant de supprimer les fichiers .class .
	
		Ainsi, pour utiliser ces commandes vous devez vous placer dans le répertoire "alnatour_hervouet" pour avoir accés.
